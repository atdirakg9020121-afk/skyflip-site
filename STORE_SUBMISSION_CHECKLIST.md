# Microsoft Store 提出チェックリスト

SkyFlip（Electron / Windows / 無料＋テーマ課金）を Microsoft Store に出すまでの手順と必要物。
自宅PCで詰まらないよう、上から順に潰せば提出できる構成。

凡例: 🏢=会社でも可 / 🏠=自宅(ソース必要) / ⏳=審査待ちが発生

---

## フェーズ0：アカウント準備 ⏳
- [ ] 🏠 **個人の Microsoft アカウント**でサインイン（※会社アカウント不可。収益は個人のもの）
- [ ] 🏠 [Microsoft Partner Center](https://partner.microsoft.com/) で開発者登録（登録料 約$19・一度きり）
- [ ] 🏠 本人確認・税務/支払い情報の登録（収益受け取りに必須。審査に数日かかる場合あり）
- [ ] 🏠 アプリ名「**SkyFlip**」を予約（Product management → 名前の予約）
      ※名前は早い者勝ち。登録できたらすぐ押さえる

## フェーズ1：パッケージ作成（appx/msix）🏠
- [ ] electron-builder を導入し、`appx` ターゲットでビルドできるよう設定
      （`package.json` の build.appx / windows 設定。identityName 等は予約後にPartner Centerの値と一致させる）
- [ ] アプリのアイコン各サイズを用意（Storeロゴ/タイル/スプラッシュ。44x44, 150x150, 310x150 など）
- [ ] ビルドした `.appx`（または `.msixupload`）を生成
- [ ] ローカルでサイドロードして起動確認（署名はStore提出時にMSが代行するため自前証明書は不要）
- [ ] バージョン番号を 1.0.0.0 形式で設定

## フェーズ2：ストア掲載情報の入力 🏢
※ 文面は `STORE_LISTING.md` にコピペ用が用意済み。
- [ ] アプリ名 / 短い説明 / 説明 を入力（→ STORE_LISTING.md）
- [ ] 主な機能・検索キーワードを入力
- [ ] **スクリーンショット**を最低1枚（推奨4〜8枚。1366x768以上）🏠
      添える説明文は STORE_LISTING.md に用意済み
- [ ] ストアロゴ画像（300x300 など）🏠
- [ ] カテゴリ：個人用設定 or ユーティリティ＆ツール
- [ ] 対応言語：日本語（英語掲載文は任意・後追いでOK）
- [ ] **プライバシーポリシーURL**：https://atdirakg9020121-afk.github.io/skyflip-site/privacy.html ✅用意済み
- [ ] サポート/問い合わせ先：contact.html の連絡先を本物に差し替えてから記入 🏠

## フェーズ3：年齢区分・申告 🏢
- [ ] 年齢別レーティングのアンケートに回答（暴力/性的表現なし → 全年齢想定）
- [ ] **データ収集の申告**が要点：
      「位置情報」を利用すること、天気取得のため外部サービスに送信することを正しく申告
      （privacy.html の内容と矛盾させない）
- [ ] 必要なアプリ機能（capability）の宣言：位置情報・インターネット

## フェーズ4：価格と提供 🏢
- [ ] 本体価格：**無料**
- [ ] 提供市場：まず日本（必要に応じて全世界）
- [ ] 公開オプション：手動公開 or 審査通過後すぐ公開 を選ぶ

## フェーズ5：アプリ内課金（テーマ）🏠 ※本体公開後でも可
- [ ] Partner Center で **Add-on（アプリ内課金アイテム）** を登録
      - 桜の舞う春 ¥320 / ネオン夜景 ¥320 / 深宇宙と銀河 ¥320 / オールテーマパック ¥980
      - 各Add-onに固有のプロダクトID（store-id）を付与
- [ ] アプリ側で購入・所有判定を実装（`StoreContext` API）
      Electronからは Node ネイティブアドオン or WinRT ブリッジ経由が必要。実装時に要調査
- [ ] 購入済みテーマのアンロック処理＆復元（別PC/再インストール時）

## フェーズ6：提出と審査 ⏳
- [ ] 全項目を埋めて「審査用に提出」
- [ ] 審査（通常数時間〜数日）。リジェクト時は理由を確認して修正・再提出
- [ ] 承認 → 公開！ Store URL を控える

## フェーズ7：公開後（サイト・告知の最終差し替え）🏢
- [ ] サイトの「Microsoft Store で入手」ボタンの href を実Store URLに（index.html ①と⑤）
- [ ] LAUNCH_POSTS.md の各原稿のURLを差し替え → X/Reddit/ProductHunt に投下
- [ ] OGP画像（assets/ogp.png）を本番に差し替え（OGP_DESIGN.md 参照）

## 自宅で最初にやるべき3つ（最短ルート）

2. アプリのスクショ／アイコン素材を書き出す
3. electron-builder で appx を一度通す（ここが技術的に一番詰まりやすい→来たら一緒に）
