# OGP画像 デザイン設計書

SNS（X / Facebook / Discord 等）でURLを貼ったときに表示されるサムネ画像 `assets/ogp.png` の設計。
自宅PCで Figma / Canva / Photopea などで作る際の指示書。

## 基本仕様
- **サイズ: 1200 × 630 px**（OGP標準。Twitter summary_large_image も同サイズでOK）
- 書き出し: PNG（写真調なら JPG も可）。ファイル名 `ogp.png`、置き場所 `assets/ogp.png`
- 文字は端から最低 60px の余白を確保（プラットフォームによる端の見切れ対策）
- 重要な文字・ロゴは中央寄りに。右下はXのUIで隠れることがあるため避ける

## カラーパレット（サイトと統一）
- 空の青（メイン）: #4a90d9
- 明るい空: #aed6f1 / #d6eaf8
- 夜・濃紺: #1f2933 / #2c3e60
- 夕焼け: #e98a5b / #f6c89a
- 文字（白）: #ffffff
- アクセント月: #f4f1e0

## フォント
- 日本語: Noto Sans JP / ヒラギノ角ゴ（太め=Bold以上）
- 英字ロゴ "SkyFlip": やや幅広のサンセリフ（Poppins / Montserrat 等）、字間を少し広げる

---

## 案A（推奨）：横グラデーションで「一日の移ろい」を一枚に
**コンセプト:** 左から右へ朝→昼→夕→夜のグラデーション。SkyFlipの世界観（時間で空が変わる）を一目で伝える。

```
┌────────────────────────────────────────────────┐
│ [朝]      [昼]        [夕]         [夜]          │ ← 横向きグラデ
│ #f9d9a0 → #4a90d9 → #e98a5b → #1f2933           │
│                                       🌙(三日月) │
│                                                  │
│            ┌──────────┐                          │
│            │  1 2 : 3 4 │  ← フリップ時計（中央） │
│            └──────────┘                          │
│                                                  │
│              S k y F l i p   ← ロゴ（白・大）     │
│        窓の外の天気を、机の上に。 ← サブコピー    │
└────────────────────────────────────────────────┘
```
- 中央にフリップ時計のモチーフ（角丸の黒パネル＋白い数字 12:34 など）
- その下に "SkyFlip"（白・72〜96px）、さらに下に小さくサブコピー
- 右上に三日月（月齢推し）、左にうっすら太陽。雲を数個ふんわり配置
- **狙い:** スクロール中でも「空・時間・時計」が即わかる。情報量と美しさのバランス◎

## 案B：夜空＋月推し（情緒・静けさ重視）
**コンセプト:** 深い夜空に本物の月。"月の満ち欠けを映す"という他にない強みを主役に。

```
┌────────────────────────────────────────────────┐
│   濃紺グラデ #1f2933 → #2c3e60                   │
│                          🌕 ← 大きめの満月       │
│       ✦      ✦                                   │
│            ┌──────────┐                          │
│            │  2 2 : 0 8 │                         │
│            └──────────┘                          │
│         S k y F l i p                            │
│   夜空の月は、今夜の本物と同じ。                  │
└────────────────────────────────────────────────┘
```
- 背景は夜のみ。星を小さく散らす
- 右上に質感のある月（クレーターの陰影を少し）
- **狙い:** 差別化ポイント（実月齢）を前面に。落ち着いた印象でブランド感が出る

## 推奨の進め方
1. まず **案C** を15分で作って暫定の `ogp.png` を置く（空欄を埋める）
2. 動画・スクショ素材が揃ったら **案A** に差し替え（本命）
3. 案Bは、月推しのキャンペーンやStore用サブ画像として転用

## 確認方法（公開後）
- X: https://cards-dev.twitter.com/validator は廃止 → 実際にポストの下書きにURLを貼ってプレビュー確認
- Facebook: https://developers.facebook.com/tools/debug/ でOGPキャッシュを更新＆確認
- 一般: https://www.opengraph.xyz/ にURLを入れて各SNSの見え方を一括プレビュー
