# CLAUDE.md — SkyFlip ショーケースサイト

## このプロジェクトは何か
デスクトップアプリ **SkyFlip** の紹介用ランディングページ（ショーケースサイト）。
アプリ本体のコードは別リポジトリ（自宅PCにある）。ここは**宣伝・集客専用のサイト**。

## SkyFlip（アプリ本体）について
- フリップ型の時計アニメーション＋現在の天気・位置情報を取得し、背景に雨・雲などのアニメを表示するデスクトップアプリ
- 技術スタック: **Electron**
- 対応OS: **Windows のみ**
- 目的: **収益化**

## 収益化の方針
- 配布の本命: **Microsoft Store**（無料アプリとして公開、署名はStoreが代行）
- 収益モデル: **無料 ＋ テーマ課金**（Store の Add-on / アプリ内課金）
  - 無料: スタンダード
  - 有料テーマ: 桜の舞う春 / ネオン夜景 / 星空と流星（各 ¥320）
  - 全部入りパック ¥980
- 課金実装は `StoreContext` API 経由（Electron からは未着手。自宅PCで実装予定）

## このサイトの構成
1ページ完結のランディングページ。上から: ①ヒーロー ②コンセプト ③特徴3つ ④テーマ紹介(課金導線) ⑤ダウンロード ⑥フッター

### ファイル
- `index.html` — 本体。日本語コピー全部入り
- `style.css` — 空・癒やし系デザイン。レスポンシブ対応済み
- `STORE_LISTING.md` — Microsoft Store 掲載文（コピペ用）
- `CLAUDE.md` — このファイル

## デザイン/トーンの方針
- コンセプト: 「情報のためでなく、眺めて心地よくなる時計」= 癒やし・リラックス系で差別化
- メインキャッチ: 「窓の外の天気を、机の上に。」
- カラー: 空の青(#4a90d9)を基調に、余白多め
