# ローンチ前 最終チェックリスト

公開（サイト・Store・告知の投下）を実行する直前に、上から順に確認する。
目的: 「本番に example@ が残る」「リンクが # のまま」のような恥ずかしい事故をゼロにする。

---

## 1. サイトのプレースホルダー差し替え漏れチェック
自宅で `grep`（または各ファイルを検索）して、以下が**残っていないこと**を確認:

- [ ] `example@example.com` が無い（contact.html）
- [ ] `legal__placeholder` の中身が実テキストに置き換わっている（privacy / terms / contact）
- [ ] `［` `］`（角括弧の仮テキスト）が本番ファイルに残っていない
- [ ] `href="#"`（空リンク）が残っていない
- [ ] 「Microsoft Store で入手」ボタンが実Store URLを指している（index.html ①ヒーロー / ⑤DL）

> まとめて確認: `git grep -nE "example@|legal__placeholder|href=\"#\"|［|］"`
> ヒットしたら差し替え忘れ。ゼロになればOK。

## 2. 画像・動画の差し替え
- [ ] `assets/hero.mp4` が本番動画になっている（仮の無し）
- [ ] `assets/hero-poster.jpg` がある（動画読込前の絵）
- [ ] `assets/ogp.png` が本番画像（OGP_DESIGN.md 参照）
- [ ] テーマ4枚のサムネが仮グラデ→実スクショに差し替わっている（style.css）
- [ ] favicon がタブに表示される

## 3. 表示確認（複数環境）
- [ ] PC のブラウザ（Chrome / Edge）で全ページ開いてレイアウト崩れ無し
- [ ] スマホ幅でも崩れない（ブラウザの開発者ツールでレスポンシブ確認）
- [ ] 全ページのフッターのリンクが正しく飛ぶ（トップ/FAQ/問い合わせ/利用規約/プライバシー）
- [ ] 動画が自動再生・ループする（muted必須。確認済みだが本番動画でも再チェック）

## 4. リンク・メタ情報
- [ ] OGP確認: https://www.opengraph.xyz/ にサイトURLを入れて、画像と説明が出る
- [ ] サイト内リンク切れが無い（faq→contact など内部リンク）
- [ ] sitemap.xml に全公開ページが載っている

## 5. Store 提出物（STORE_SUBMISSION_CHECKLIST.md と連動）
- [ ] スクショ最低1枚（推奨4〜8）アップ済み
- [ ] プライバシーポリシーURL を記入済み（https://atdirakg9020121-afk.github.io/skyflip-site/privacy.html）
- [ ] データ収集（位置情報）の申告が privacy.html と矛盾しない
- [ ] 価格=無料、提供市場を選択済み
- [ ] 課金テーマ（Add-on）の価格がサイト/STORE_LISTINGと一致

## 6. 告知文（LAUNCH_POSTS.md）
- [ ] すべての原稿の URL を実サイト/実Store URLに差し替え済み（`［公開後にStore URLを記入］`が残っていない）
- [ ] X 用に動画 or GIF を用意（テキストだけは伸びない）
- [ ] Reddit: 投稿先 sub のセルフプロモルールを確認
- [ ] Product Hunt: 投下は平日の米国太平洋時間 0:01 が定番。最初のコメントを用意

## 7. 公開直後の見回り
- [ ] 公開URLが 200 で開ける（`curl -I` でも可）
- [ ] Store ページが実際に表示され、ダウンロードできる
- [ ] エゴサ用に「SkyFlip」でX検索を保存しておく（初速の反応を拾う）

---

## ひとことルール
**「仮のもの」を本番に出さない。** 仮テキスト・仮画像・空リンクは、上の grep で必ず炙り出してから公開する。
